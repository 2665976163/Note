# Redis

​			Redis是一个key-value形式的存储系统， 它支持存储的value类型有string(字符串)、list(链表)、set(集合)、zset(sorted set --有序集合)和hash（哈希类型）等一些其他类型，这些**数据类型**都支持**CRUD**及取交集并集和差集及更丰富的操作，而且这些操作都是原子性的.

 **Redis：**读的速度是110000次/s,写的速度是81000次/s
